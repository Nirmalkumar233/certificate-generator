## Setup Steps

#### Using Docker
- Make sure you have docker and docker-compose installed
```
$ docker-compose up
```

#### Manually
- Go to directory
```
$ cd certificate-generator
```
- Install Dependencies
```
$ npm install
```
- Start LocalHost Server
```
$ npm run start
```

## Tech Stack Used

  - React
  - Firebase
  - Material UI
